# Search Algorithm Visualiser

A visual implementation of BFS, DFS, and A* pathfinding algorithms built in Java.

## What it does
- Renders an interactive grid where you can set start, end, and wall nodes
- Visualises how BFS, DFS, and A* explore the grid differently
- Shows the final path found by each algorithm

## Algorithms implemented
- **BFS** - explores all neighbours equally, guarantees shortest path
- **DFS** - explores deep before backtracking, not always shortest
- **A*** - uses a heuristic to explore smarter, fastest to find optimal path

## Tech stack
Java · OOP · Swing (GUI) · Data Structures

## How to run
```bash
git clone https://github.com/goutami-15/search-algorithm-visualiser
cd search-algorithm-visualiser
javac src/*.java
java src/Main
```
